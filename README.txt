Mod Information
-------------------------------------------
Downloads for versions 1.7 - 1.10 can be found here:
http://mods.curse.com/mc-mods/minecraft/247205-fbtoggle#t1:other-downloads

Minecraftforums thread found here, for bug reports or more information about mod:
http://www.minecraftforum.net/forums/mapping-and-modding/minecraft-mods/2707208-fullbright-toggle-mod-1-7-1-10

Mod Installation Guide:
Download mod version corresponding to the Minecraft version you are using, and
drop it in the mods folder of your .minecraft files. You'll need to be using
Forge in order to use this mod.
